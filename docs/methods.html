Methodology and sources

How this dashboard was built

Project question

Can a citizen-facing dashboard make Hunting Bayou waterway observations easier to understand by translating near-live USGS station data into plain-language context and a Dutch-informed resilience framing?

Why Hunting Bayou?

Hunting Bayou is the only corridor fully implemented in this version. It is an under-discussed Houston corridor with USGS monitoring stations, identified local flood-control projects, and an urban watershed context near Herman Brown Park.

Dutch design translation

The project emphasizes monitoring, maintaining normal use where possible, and interpreting changing water conditions. That translation is expressed by showing observed gage height, discharge, rate changes, and the intended role of detention and conveyance improvements without claiming operational effectiveness.

Official data sources

  • USGS Water Services for instantaneous values and historical event screening.
  • Harris County Flood Control District project pages for local infrastructure context.
  • Harris County Flood Warning System for official flood alerts.

Derived metrics and definitions

  • Gage height: water-surface elevation above a reference point, shown in feet.
  • Discharge: volume of water passing the station per second, shown in cubic feet per second.
  • One-hour and six-hour change: difference in gage height compared with the same station one or six hours earlier.
  • Max observed 24h: highest station reading seen in the past 24 hours.
  • Trend label: Rising, Falling, Relatively Stable, or Insufficient Data based on recent stage slope.
  • Freshness: Recent (<= 60 min), Delayed (<= 180 min), Stale (> 180 min).

Limits of interpretation

This dashboard does not predict flooding, certify infrastructure performance, or replace official alerts. It does not compute flood risk, capacity remaining, or detention effectiveness without official thresholds or cited analysis.

Refresh process

A scheduled GitHub Actions workflow runs hourly at minute 17 to fetch USGS data, process outputs, and update the static site. The build is best-effort and the dashboard labels near-live observation age.

Low-power display output

A single-file HTML snapshot is generated for conceptual offline or low-power display use. It is designed to be self-contained and suitable for display concepts like an e-ink or Raspberry Pi viewer.

Sources: USGS Water Services, HCFCD project pages, Harris County Flood Warning System.
